# 📖 Book Vision

Book Vision เป็นเครื่องมือจำลองการเขียนหนังสือ ที่ช่วยให้นักเขียนและสำนักพิมพ์สามารถวางแผนและประเมินความคืบหน้าของงานเขียนได้อย่างแม่นยำ โดยการจำลองลักษณะทางกายภาพของหนังสือและเป้าหมายการเขียน

link : https://book-vision-dd23lwpk6-chantip-pis-projects.vercel.app/
---

## ✨ คุณสมบัติหลัก
- กำหนด **ขนาดกระดาษ** (เช่น B5, A4)
- เลือก **ฟอนต์และขนาดตัวอักษร**
- ปรับ **ระยะห่างบรรทัด** และ **น้ำหนักกระดาษ**
- ตั้งค่า **เป้าหมายการเขียน** เช่น จำนวนหน้า, จำนวนตัวอักษรทั้งหมด, เป้าหมายตัวอักษรต่อวัน
- แสดงผลการจำลอง:
  - จำนวนคำทั้งหมด
  - ความคืบหน้า (%)
  - จำนวนวันที่ต้องใช้
  - ตัวอักษรต่อหน้า, บรรทัดต่อหน้า, คำต่อหน้า
  - น้ำหนักและความหนาของหนังสือที่จำลอง

---

## 🖥️ วิธีการใช้งาน
1. เปิดหน้าเว็บแอปพลิเคชัน Book Vision
2. กำหนดค่าหนังสือในส่วน **ตั้งค่าหนังสือ**
3. ใส่เป้าหมายการเขียน เช่น จำนวนหน้าและตัวอักษรต่อวัน
4. ตรวจสอบผลการจำลองในส่วน **ผลลัพธ์การจำลอง**
5. ใช้ข้อมูลเพื่อวางแผนการเขียนและติดตามความคืบหน้า

---

## 📊 ตัวอย่างผลลัพธ์
- เป้าหมาย: 200 หน้า, 10,000 ตัวอักษร
- ความคืบหน้า: 2.5%
- จำนวนวันที่ต้องใช้: 401 วัน
- น้ำหนักหนังสือจำลอง: 8 กรัม
- ความหนา: 0.3 มม.

---

## 🚀 เหมาะสำหรับใคร
- นักเขียนที่ต้องการติดตามความคืบหน้าของงานเขียน
- สำนักพิมพ์ที่ต้องการประเมินลักษณะทางกายภาพของหนังสือ
- ผู้ที่สนใจการวางแผนการเขียนเชิงปริมาณ
